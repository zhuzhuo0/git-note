## Typescript 实战指南笔记

#### 第1章

```
# 安装typescript
yarn global add typescript
# typescript自带的tsc命令无法直接运行ts文件，通常还要装ts-node，提供直接运行typescript的能力
yarn global add ts-node
# 直接运行文件
ts-node demo.ts

# 安装shell命令
cmd/ctrl + shift + p 
# 输入 install 'code'
# 选择Shell Command :install 'code' command in PATH
code ./folder   code test.js  可以快速打开文件或文件夹
```
